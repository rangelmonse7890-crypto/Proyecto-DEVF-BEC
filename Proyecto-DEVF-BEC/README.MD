## Proyecto final del modulo 1.

El propósito de esta página web es mostrar un blog de curiosidades de los conejos donde los usuarios puedan leer todos sobre los conejos, explorar los tipos de razas contenido visual atractivo. Además, funciona como un proyecto educativo para practicar la creación y diseño de una página web en HTML.

## Objetivo
Crear una página web tipo blog de conejos que permita compartir informacion visual de forma atractiva, sencilla y organizada.

## Tecnologías usadas

-   **HTML** – Para la estructura principal del sitio.   
-   **CSS** – Para los estilos, diseño visual y maquetado.    
-   **JavaScript** – Para funciones interactivas como agregar entradas, buscador y almacenamiento local.
-  
## Estructura del proyecto 
El proyecto incluye:
    -   Título      
    -   Imagen        
    -   Descripción        
    -   Fecha      
    -   Contenido completo  
        Esta sección permite que cada publicación tenga su propio espacio y datos organizados.
       
## Cómo ejecutar el proyecto
-   Descarga o copia el archivo **index.html**.   
-   Guarda el archivo en cualquier carpeta de tu computadora.   
-   Ábrelo dando doble clic; se abrirá en tu navegador (Chrome, Edge, Firefox, etc.).  
-   ¡Listo! El blog funciona sin internet y sin instalar nada porque usa HTML, CSS y JavaScript directamente.
